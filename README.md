## Memory Game

A responsive memory card game built with React and Vite.

## Features

Click-based memory gameplay
Randomized card shuffle after every selection
Current score tracking
Best score tracking
Streak tracking system
Dynamic card/image rendering from an API
Responsive layout for desktop and mobile devices
Restart game functionality
Built With
React
Vite
JavaScript
CSS
Project Structure
components/ — reusable React components
styles/ — CSS files
App.jsx — main application logic
main.jsx — React application entry point
Notes

Cards are shuffled using the Fisher–Yates shuffle algorithm to ensure unbiased randomization.

## Installation

npm install
